## Список пользователей на Vanilla JS

Приложение по тестовому заданию, которое отображает список пользователей, позволяет фильтровать их по возрасту, загружать фотографии и сортировать таблицу. Выполнено на чистых **HTML, CSS и JavaScript** (без сборщиков и фреймворков).

### Как запустить локально

1. **Перейдите в корень проекта**
2. **Установите Node.js** (если ещё не установлен)
3. В терминале выполните команду: `npx serve .`
4. Откройте в браузере адрес, который выведет сервер (например, `http://localhost:3000`).

### Структура проекта (для навигации по коду)

- **Корень проекта**

  - `index.html` — входная HTML‑страница, подключение CSS и JS.
  - `users.json` — исходный список пользователей (имя, фамилия, возраст, email, id и т.п.).
  - `README.md` — описание задания и проекта.

- **Папка `styles/`**

  - `base.css` — базовая типографика, контейнер, общая вёрстка.
  - `controls.css` — стили панели фильтров, кнопок и элементов управления.
  - `table.css` — стили таблицы с пользователями, состояния пустого списка и адаптивная таблица.

- **Папка `components/` (HTML‑шаблоны)**

  - `users-toolbar.html` — панель фильтрации по возрасту.
  - `users-list.html` — таблица со списком пользователей и заголовками для сортировки.

- **Папка `js/` (JavaScript‑логика)**

  - `main.js` — точка входа: загрузка layout’ов, инициализация фильтра, сортировки, рендера и AJAX‑загрузки.
  - `api.js` — загрузка пользователей (`users.json`) и установка границ возраста.
  - `dom.js` — вспомогательные функции для поиска DOM‑элементов (tbody, empty state и пр.).
  - `config/constants.js` — константы путей (`PATHS`) и идентификаторов DOM (`DOM_IDS`).

- **Папка `js/features/users/`**

  - `render.js` — отрисовка строк таблицы, создание ячеек, подключение аватаров.
  - `dataProcessing.js` — применение фильтра по возрасту и сортировки к массиву пользователей.
  - `ageFilter.js` — инициализация слайдера возраста и связь с состоянием.
  - `tableSort.js` — логика UI кнопок сортировки и синхронизация с состоянием.
  - `state/core.js` — объект `STATE` (пользователи, возрастной фильтр, сортировка).
  - `state/users.js` — установка списка пользователей и обновление фото пользователя.
  - `state/filters.js` — состояние фильтров и сортировки (границы и текущие значения).
  - `state/storage.js` — сохранение/загрузка пользователей с фото в/из `localStorage`.

- **Папка `js/components/`**
  - `avatar/avatar.js` + `avatar/avatar.css` — компонент аватара, загрузка/удаление фото, отображение инициалов.
  - `range/range.js` + `range/range.css` — компонент диапазонного слайдера возраста.

### Как пользоваться приложением

- **Фильтрация по возрасту**

  - В верхней панели перетащите ползунки возраста (минимальный и максимальный значения).
  - Таблица обновляется по мере изменения диапазона.

- **Загрузка/изменение фото пользователя**

  - Наведите курсор на ячейку с аватаром (или тапните на мобильном), нажмите на иконку **«Загрузить фото»**, выберите файл.
  - Чтобы удалить фото, нажмите на иконку **«Удалить фото»** — вернётся аватар с инициалами.

- **Сортировка списка**
  - Нажмите на заголовок **Фамилия** или **Возраст**:
    - 1‑й клик — сортировка по возрастанию,
    - 2‑й клик — по убыванию,
    - 3‑й клик — без сортировки.
