## Список пользователей на Vanilla JS

Это тестовое задание: список пользователей, реализованный на чистых **HTML, CSS и JavaScript**, без сборщиков, фреймворков, TypeScript и т.д.

### Функциональность

- **Список пользователей**: аватар, имя, фамилия, возраст и email.
- **Фото пользователей**:
  - отображение аватара (фото или инициалы),
  - загрузка фотографии для каждого пользователя,
  - сохранение фото и данных в `localStorage`.
- **AJAX-загрузка данных** из файла `users.json`.

### Структура проекта

- `index.html` — разметка страницы и подключение стилей/скриптов.
- `style.css` — стили и адаптивный дизайн.
- `app.js` — логика отображения, фильтрации, сортировки и загрузки фото.
- `users.json` — данные о пользователях.
- `public/favicon.ico` — иконка сайта.

### Как запустить локально

Нужно запустить простой статический сервер:

- Установить Node.js
- Выполнить в команду `npx serve .`

После запуска откройте в браузере адрес, который выведет сервер (например, `http://localhost:3000`).
